# Chronicle

A daily journaling app with a social accountability layer.

Built after 9 months of personal journaling that changed everything.

**Live:** [mychronicle.dev](https://mychronicle.dev)

---

## What it is

Chronicle is a minimal daily journal. One entry per day, no more. 
Follow others and see their entry after you've posted yours — 
accountability without pressure.

---

## Features

- One entry per day, resets at midnight
- Personal journal archive
- Follow others, see their daily entry on the feed
- Streak tracking
- Installable PWA — works on mobile like a native app

---

## Stack

- React + Vite
- Supabase — auth, database, row level security
- Tailwind CSS
- Deployed on Vercel
